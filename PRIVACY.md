# Privacy Policy

**Last Updated:** December 14, 2025

## Overview

Knowledge Pro is a Dify plugin tool that enables management of Dify Knowledge Bases via the Dify API. This privacy policy explains how data is handled when using this plugin.

## Data Collection

### Credentials
- **API Key**: Required for authentication with the Dify API
- **Base URL**: Required to connect to your Dify instance

These credentials are stored securely within the Dify platform and are only used to authenticate API requests to your Dify instance.

### Operational Data
The plugin processes the following types of data as part of normal operation:
- Dataset/Knowledge Base information
- Document content and metadata
- Chunk/Segment content
- Search queries for retrieval operations

## Data Processing

### How Data is Used
- Credentials are used solely for authenticating API requests to your Dify instance
- All data operations are performed through the Dify API
- No data is stored by the plugin itself; all data resides in your Dify instance

### Data Transmission
- All API communications use HTTPS encryption
- Data is transmitted directly between the plugin and your Dify instance
- No data is transmitted to third-party services

## Third-Party Services

This plugin interacts exclusively with the Dify API. No other third-party services are used.

For Dify's privacy policy, please refer to the official Dify documentation.

## Data Retention

- The plugin does not retain any user data
- All data is stored and managed by your Dify instance
- Credential storage is handled by the Dify platform

## User Rights

As a user, you have the right to:
- Remove the plugin at any time
- Revoke API keys to disable access
- Delete any data from your Dify Knowledge Bases

## Security

- API keys are treated as sensitive information and stored securely
- All API requests are authenticated using bearer token authentication
- HTTPS is used for all communications

## Changes to This Policy

This privacy policy may be updated periodically. Changes will be reflected in the "Last Updated" date.
